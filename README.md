# 🤖 Transformable Robot Scene

A project developed for the **Computer Graphics** course.

## 📖 Overview

This project implements an interactive 3D scene using Three.js.

The scene features a transformable robot that can switch between robot and truck modes, a detachable trailer, multiple camera views, simple animations and collision detection.

The project explores geometric modeling, hierarchical transformations, camera systems, animation techniques and interactive user controls.

## ✨ Features

* 🤖 Transformable robot model
* 🚛 Trailer model and movement
* 🎥 Multiple camera views
* 🔄 Robot-to-truck transformation
* ⚙️ Hierarchical object transformations
* 📦 Primitive-based 3D modeling
* 🎬 Animation update loop
* 💥 Collision detection using AABBs
* 🕸️ Wireframe and solid rendering modes

## 🛠️ Built With

* JavaScript
* Three.js

## 🎮 Controls

### Cameras

* `1` → Front view
* `2` → Side view
* `3` → Top view
* `4` → Perspective view

### Robot Transformation

* `Q / A`
* `W / S`
* `E / D`
* `R / F`

### Trailer Movement

* Arrow Keys

### Rendering Mode

* `7` → Toggle Wireframe / Solid
